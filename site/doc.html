1: Создание загрузочной флешки

1.1: Загрузка iso образа Rubin

Перейдите на страницу "Загрузки" и скачайте iso образ, для вашей архитектуры.

1.2: Создание загрузочной флешки, используя Balenta Etcher

Загрузите и установите Balenta Etcher. Выберите iso файл Rubin, и диск, который вы хотите использовать, как установочную флешку. Начните запись iso файла. Узнайте модель своего wifi адаптера и производителя своей материнской платы. Эта информация потребуется вам, для дальнейшей установки системы.

1.3: Создание загрузочной флешки, используя Ventoy

Загрузите и установите Ventoy. Откройте программу, выберите диск, который вы хотите использовать как загрузочную флешку и установите на него Ventoy. Затем переместите iso образ Rubin на диск Ventoy.

1.4: Создание загрузочной флешки вручную

Сформатируйте свою флешку в формат FAT32. Для этого можно использывать стороние программы. Например, Active@ Disk Image. Затем с помощью программы-архиватора (Например WinRar.) распакуйте iso образ на флешку.

2: Установка

2.1: Загрузка в систему

Перезагрузитесь в bios. (Управление в bios и boot меню осуществляется стрелками, для выбора системы нажмите Enter.) Выберите вашу загрузочную флешку в boot меню. Дождитесь загрузки системы и начните установку, выбрав пункт "Try or install Rubin". Затем перед вами откроется графический установщик.

2.2: Начало графической установки

В меню слева, выберите русский язык, и начните установку. По желанию, (Если на данном этапе у вас работает интернет) установите обновления и драйвера. Следуйте дальнейшим этапам графического установщика, завершите установку системы и перезагрузите компьютер. Затем переходите к шагу №3.

3: Установка драйверов Nvidia

3.1: Установка драйверов на видеокарту, используя ubuntu-drivers

Откройте терминал одновременным нажатием клавиш control+alt+t, и введите команду "sudo ubuntu-drivers autoinstall", а затем введите пароль (Пароль может не отображаться, это нормально.) Эта команда установит последние драйвера из официального репозитория Ubuntu. Перезагрузите компьютер.

3.2: Установка драйверов на видеокарту, используя nvidia-detect

Установите nvidia detect командой "sudo apt install nvidia-detect". Введите команду nvidia-detect. На экране высветиться название нужного драйвера. Установите драйвер командой "sudo apt install название драйвера". Перезагрузите компьютер.

3.3: Установка драйверов с официального сайта Nvidia

Перейдите на сайт Nvidia и загрузите нужную версию драйвера. Переместите драйвер в папку /home/имя вашего пользователя. Впишите в терминал команды: "sudo su", "nano /etc/modprobe.d/blacklist-nouveau.conf" в открывшимся файле, впишите следующий текст:

blacklist nouveau options nouveau modeset=0

Сохраните текст, нажав control+x, затем впишите "y" и два раза нажмите enter. Выполните команду "update-initramfs -u", "systemctl set-default multi-user.target", "systemctl reboot". Перед вами откроется черная командная строка, впишите имя вашего пользователя и нажмите enter. Затем ваш пароль и нажмите enter. Введите команду "sudo su", Чтобы посмотреть имя драйвера, воспользуйтесь командой "ls", для отображения имён всех файлов в папке, затем команду "bash имя вашего драйвера". Во время установки выберите следующие опции:

  • The CC version check failed: Ignore CC version check
  • Install NVIDIA's 32-bit compatibility libraries: Yes
  • An incomplete installation of libglvnd was found. Do you want to install a full copy of libglvnd? This will overwrite any existing libglvnd libraries: Install and overwrite existing filesort installation
  • Would you like to run the nvidia-xconfig utility to automatically update your X configuration file so that the NVIDIA X driver will be used when you restart X?  Any pre-existing X configuration file will be backed up: Yes

Введите команду "systemctl set-default graphical.target", а затем "systemctl reboot".

4: Удаление лишних систем из boot меню

Если после установки в вашем boot меню остались несуществующие системы, вы можете удалить их с помощью утилиты efibootmgr. Выполните команду "efibootmgr", для отображения списка систем. Перед вами высветиться список входов из вашего boot menu.

Чтобы удалить ненужные входы, введите команду "sudo efibootmgr -Bb номер boot order, который высвечивается у нужного входа в систему". Можете удалить все входы КРОМЕ UBUNTU.

5: Добавление и настройки русской раскладки

Откройте настройки клавиатуры (Система -> Параметры -> Оборудование -> Клавиатура), перейдите во вкладку "Раскладки", добавьте русскую раскладку, перейдите в параметры и выберите любую удобную вам горячую клавишу для смены раскладки клавиатуры.
